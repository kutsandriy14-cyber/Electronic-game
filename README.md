# ЭлектроСимулятор Ядерного Реактора (ElectroSim) — Панель Управления и Разработки

Добро пожаловать в проект **ElectroSim** — продвинутый интерактивный симулятор физических процессов, цепей и термоядерной активности.

---

## 🛠 Актуальный релиз и Сборка
*   **Свежая отладочная сборка приложения**: Находится в корне репозитория проекта под именем [**`app-debug.apk`**](./app-debug.apk). Она полностью пересобрана на актуальной версии кода.
*   **Текущая версия приложения**: `1.4.23` (код версии: `10423`).

---

## 📝 Журнал Изменений (Changelog) — Текущий Этап Разработки

### 🤖 Автоматизация, CI/CD и Процессы Сборки
*   **Автоматическое создание GitHub-релизов**: Создан workflow GitHub Actions (`.github/workflows/android.yml`), который при пуше изменений в ветки `main`/`master`/`dev` автоматически выполняет релиз на GitHub, собирает актуальный APK-файл и прикрепляет его к релизу.
*   **Автоматический инкремент версии (Auto-Bump)**: Настроен скрипт автоматического повышения минорной версии (третьей цифры versionName) и versionCode в файле `app/build.gradle.kts` при каждом коммите/релизе через CI.
*   **Решение проблемы KSP в CI**: Исправлен критический сбой компиляции процессора аннотаций Kotlin Symbol Processing (KSP) (`java.lang.NullPointerException: Cannot invoke ... Application.getService ... because ... is null`) в среде GitHub Actions путем активации стандартного процесса сборки с использованием Gradle Daemon.
*   **Удобство дистрибуции**: APK-файл компилируется и помещается прямо в корень проекта под именем `app-debug.apk` для мгновенного доступа и загрузки.

### ⚛️ Симуляция и Физический Движок
*   **Сетка Суперкомпьютера 32k**: Настроена гигантская координатная сетка размером **180x180** для симуляции сверхсложных физических структур.
*   **Улучшенная термоядерная симуляция**: Оптимизирована симуляция реакций урана, достижение критического состояния реактора, а также физика эффекта Лейденфроста (Leidenfrost-эффективность воды).
*   **Физика Кориума**: Переработана формула формирования и растекания расплавленного ядерного сырья (кориума) при перегреве топливных элементов.
*   **Очистка физического мира**: Добавлена корректная поддержка удаления двойных дверей, силовых барьеров и других сложных структурных компонентов симулятора без повреждения сетки.
*   **Исправление критических багов**: Исправлена ошибка сборщика, возникавшая при неявном приведении типов внутри логики распределения вероятностей `flowChance`.

### 📺 Пользовательский Интерфейс (UI/UX)
*   **Контроль Телеметрии**: Добавлена новая многофункциональная интерактивная панель телеметрии с поддержкой полного скрытия и сворачивания в суперкомпактный минималистичный оверлей для освобождения полезной площади экрана.
*   **Стилевые правки**: Отрегулирована контрастность Material 3 элементов управления симуляции.

## 🚀 Как запустить сборку вручную
Если вам требуется пересобрать проект локально:
1. Убедитесь, что у вас установлен JDK 17.
2. Выполните команду сборки:
   ```bash
   gradle assembleDebug
   ```
3. Готовый APK-файл будет помещен в корневую директорию проекта.
